# Codacade — Retro 80s Web Arcade

A retro 80s-themed web arcade featuring classic and modern mini-games with neon Outrun/synthwave aesthetics.

## How to Run

```bash
cd Projects/codacade
npx -y serve .
```

Then open http://localhost:3000 in your browser.

## Main Files

- `index.html` — Single-page app shell
- `css/style.css` — Retro design system
- `js/app.js` — SPA router and game registry
- `js/auth.js` — JWT auth (client-side prototype)
- `js/leaderboard.js` — Score tracking and leaderboard
- `games/hangman/` — Hangman game
- `games/star-blaster/` — Star Blaster (Galaga-style) game

## APIs Used

- None (fully client-side prototype)
